# Sections

This file defines the rule categories, ordering, impact levels, and prefix mapping for the Blode Icons React skill.

---

## 1. Public API and naming (api)

**Impact:** CRITICAL
**Description:** Invalid import paths, wrong module shapes, or bad naming guidance will break consumer code and docs examples immediately.

## 2. Docs and quick starts (docs)

**Impact:** HIGH
**Description:** Install snippets and onboarding examples are the fastest path users copy, so drift here spreads bad usage patterns quickly.

## 3. Repo workflow and releases (repo)

**Impact:** HIGH
**Description:** This monorepo has package-specific release behavior and changeset requirements that are easy to miss when working from subdirectories.
